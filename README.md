# 🤖 بوت إدارة تيليجرام

## 📦 التثبيت

```bash
pip install python-telegram-bot==20.7
```

## ⚙️ الإعداد

1. افتح ملف `bot.py`
2. غيّر `YOUR_BOT_TOKEN_HERE` بالتوكن بتاعك من [@BotFather](https://t.me/BotFather)
3. غيّر `KICK_BAN_LIMIT = 20` لو عايز تغيّر الحد (افتراضي 20)

## 🚀 التشغيل

```bash
python bot.py
```

## 📋 الأوامر

| الأمر | الوصف |
|-------|-------|
| `/promote` | رفع مشرف مع تحديد صلاحياته |
| `/demote` | نزول مشرف |
| `/kick` | طرد عضو |
| `/ban` | حظر عضو |
| `/stats` | إحصائيات الطرد/الحظر |

## 🎯 طرق الاستخدام

كل الأوامر تشتغل بـ 3 طرق:

**1. بالرد على رسالة:**
```
/promote (رد على رسالة الشخص)
```

**2. باليوزرنيم:**
```
/promote @ahmed
```

**3. بالـ ID:**
```
/promote 123456789
```

## 🛡 صلاحيات المشرف (عند /promote)

بيظهر ليك كيبورد تختار منه:
- ✅ حذف الرسايل
- ✅ حظر الأعضاء  
- ✅ تثبيت الرسايل
- ✅ تغيير المعلومات
- ✅ إضافة مشرفين
- ✅ الدعوة بلينك

## 🚫 فلتر الكلمات

- يشتغل تلقائياً على كل الرسايل
- المشرفين مش بيتطبق عليهم الفلتر
- بيحذف الرسالة ويبعت تحذير يختفي بعد 5 ثواني
- بيشيل الكلمة حتى لو كانت في النص مش كلمة لوحدها

## ⚠️ نظام الطرد التلقائي

- لو مشرف طرد أو حظر أكتر من 20 عضو → ينزل من الإشراف تلقائياً
- الجروب بياخد إشعار بده
- تقدر تغيّر الحد من `KICK_BAN_LIMIT`

## 📝 إضافة كلمات جديدة للفلتر

افتح ملف `bad_words.py` وضيف الكلمة في القايمة:
```python
BAD_WORDS = [
    "كلمتك الجديدة",
    ...
]
```

## 🔧 متطلبات البوت في الجروب

- لازم تضيف البوت كمشرف
- لازم يكون عنده صلاحية حذف الرسايل وحظر الأعضاء
